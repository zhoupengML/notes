* emacs 常用命令
** Insert piecewise function
#+BEGIN_SRC 
\begin{equation}
 P_{r-j}=
 \begin{cases}
 0 &\mbox{if $r-j$ is odd}\\
 r!\,(-1)^{(r-j)/2} &\mbox{if $r-j$ is even}
 \end{cases}
\end{equation}
#+END_SRC

** Insert footnote [fn:1]
   - C-c C-x f
   - C-c C-c : jump to the footnote location
** 插入时间
#+BEGIN_SRC 
C-u M-! date
#+END_SRC
** 显示图片
#+BEGIN_SRC 
C-c C-x C-v
#+END_SRC
** 打开当前链接
#+BEGIN_SRC 
C-c C-o
#+END_SRC
